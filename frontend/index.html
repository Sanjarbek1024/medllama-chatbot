Model Specifications

🤖
Base Model
TinyLlama 1.1B Chat v1.0
TinyLlama/TinyLlama-1.1B-Chat-v1.0 — a compact but capable language model with 1.1 billion parameters, optimized for chat-style conversations.
🧬
Training Dataset
MedQuAD
Medical Question Answering Dataset — a curated collection of real medical Q&A pairs from trusted health sources like NIH, CDC, and MedlinePlus.
⚡
Fine-tuning Method
QLoRA (4-bit Quantization)
Quantized Low-Rank Adaptation — an efficient fine-tuning technique that reduces memory usage by 4× while preserving model quality. Ideal for training on limited hardware.
📊
Training Size
2,000 Q&A Pairs
The model was fine-tuned on 2,000 medical question-answer pairs, teaching it to respond accurately to health-related queries in a conversational format.
